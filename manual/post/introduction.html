Introduction

In most cases where the Discrete Element Method (DEM) is used, it is necessary to carry out some averaging of the data to a scale more representative of what is required. As such it is quite common for spatial averaging, temporal averaging, or both, to be applied to the DEM data. To aid the process, a toolbox for the processing of DEM data both temporally and spatially has been implemented at Particle Analytics. The toolbox has been developed to support many software packages (such as EDEM, PFC, DEMPack and LAMMPS) and provides a simple interface for the averaging process for the large datasets that DEM simulations produce.
